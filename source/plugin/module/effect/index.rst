=================
module-effect
=================

这部分作为全平台粒子绘制工具，提供了大量基本函数。

**该模块依赖于：**

* common
* common-5

**安装方式如下所示：**

.. code-block:: kotlin

    taboolib {
        install("common", "common-5", "module-effect")
    }
